필터 입력 시 사용할 수 있는 문법 목록입니다. ***라이브 테일***과 ***로그 검색***은 동일한 문법을 사용합니다.

|문법|예시|설명|
|----|----|----|
|include(+)|`+ote-gateway`|`ote-gateway`를 포함한 것만 필터링|
|^|`@txid:+-6611219762256426545`|`@txid`가 `-6611219762256426545`인 것만 필터링|
|exclude(-)|`-ote-gateway`|`ote-gateway`를 제외한 것만 필터링|
|startWith(*)|`ERROR*`|`ERROR`로 시작하는 것만 필터링|
|endWith(*)|`*ERROR`|`ERROR`로 끝나는 것만 필터링|

:::note

include(+) 문법 사용 시 '`-`1234'와 같이 음수를 포함하는 문자열을 검색하려면 '`+`'기호를 추가해 '`+-`1234'로 검색하세요.

:::
