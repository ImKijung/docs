토폴로지 기능을 이용하기 위해 에이전트에 옵션을 적용하는 방법을 제공합니다.

## 적용 가능 에이전트 및 버전

* Java 에이전트 1.7.1 버전 이상
* Node.js 에이전트 0.4.1 버전 이상

## whatap.conf 설정

다음은 토폴로지 표현을 위해 *whatap.conf* 파일에 설정할 수 있는 정보 수집 옵션입니다.

{@include: ../apm/_common-topology-map.mdx}

그룹 및 통합 토폴로지 사용을 위해 다음과 같이 옵션을 추가하세요.

```sh title='Java : JVM 옵션 추가'
-Dwhatap.okind={그룹 식별자}
```

```javascript title='Node.js : env 추가'
process.env.WHATAP_OKIND = '{그룹 식별자}'
```
