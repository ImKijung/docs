사용자 인터페이스(UI) 소개

초기 화면​

와탭 모니터링 서비스에 로그인 시 나타나는 초기 화면의 요소들을 소개합니다. 초기 화면에 나타나는 메뉴는 프로젝트 목록입니다.

고정 메뉴 영역

  • 메뉴 : 왼쪽에는 사용자가 진입한 메뉴 경로를 표시합니다.

  • 테마 : 현재 페이지의 테마를 변경할 수 있습니다. 기본값은 Light 모드입니다.

    • Light : 기본 적용된 테마입니다.
    • Dark : 검은 배경의 테마를 적용합니다.
    • Presentation : 빔 프로젝트를 이용할 경우 최적화된 색상을 적용합니다.
    • Add Custom Theme : 사용자가 원하는 색상을 설정할 수 있습니다.

  • 고객 지원 / 설정 및 정보

    온라인 고객 지원을 받을 수 있습니다. 새 문의하기 버튼을 선택해 기술 및 비용/결제 등에 관한 문의를 할 수 있습니다. 이미 상담한 내용은 상담목록에서 확인할 수 있습니다. 유선 문의를 하려면 을 선택하세요.

    문의 내용에 대한 언어, 알림, 광고 수신 등을 설정하려면 을 선택하세요.

  • 프로필 아이콘 : 가장 오른쪽에 위치한 아이콘을 선택하세요. 계정 정보 관리, 조직 생성, 조직 관리, 사용자 언어를 선택할 수 있는 기능을 제공합니다. 로그아웃을 선택하면 이용 중인 서비스 화면에서 로그아웃합니다.
