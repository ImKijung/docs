1. 라이선스를 발급 받으면 ***와탭 에이전트 파일 다운로드*** 섹션에서 ***Download*** 버튼이 활성화됩니다. 버튼을 클릭해 와탭 에이전트 파일을 다운로드하세요.

    :::tip
    서버에서 직접 다운로드할 수도 있습니다.
    ```bash
    wget https://service.whatap.io/agent/whatap.agent.java.tar.gz
    ```
    :::

1. 모니터링 대상 서버에 접속하세요.
1. 원하는 위치에 에이전트 파일의 압축을 해제하세요. 이 위치가 ``\$WHATAP_HOME``이 됩니다.

    ```bash
    tar -zxvf whatap.agent.java.tar.gz
    ```
