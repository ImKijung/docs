프로젝트 그룹에 관한 자세한 내용은 ~~다음~~에서 확인하세요.

1. 모니터링 프로젝트를 생성하려면 왼쪽 메뉴에서 ***+ Project*** 버튼을 클릭하세요.
1. 프로젝트 이름을 입력한 다음 사용할 플랫폼을 선택하세요.
1. 가까운 데이터 서버 지역(AWS 또는 Azure)을 선택하세요.
1. ***타임 존***을 선택하세요.<br/>기본 값은 (GMT +9:00) Seoul, Tokyo, Yakutsk입니다.
1. ***저장*** 버튼을 클릭하세요.

:::note
* 데이터 서버 지역은 리전을 의미합니다.
* 클라우드에서 리전이란 클라우드 서비스를 제공하기 위해 설치하는 데이터 센터의 묶음입니다.
* 특정 리전을 선택하면 해당 리전에 속해 있는 데이터 센터에 사용자의 데이터를 저장합니다.
:::
