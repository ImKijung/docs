1. 적용 대상(WAS 및 Batch, Daemon과 같은 Java 애플리케이션)에 javaagent 옵션으로 whatap agent path를 추가합니다.
    ```bash title=JVM_OPTS
    -javaagent:/whatap/whatap.agent-#.#.#.jar
    ```
2. 서버를 다시 시작하세요.
3. 프로세스 정보에서 설정한 옵션을 확인했다면 설정 적용을 완료한 것입니다.

![ps -ef|grep java](https://img.whatap.io/media/agent_java/install/psgrep.png)

환경별 설정 위치는 ~~설치하기~~ 페이지에서 확인할 수 있습니다.
