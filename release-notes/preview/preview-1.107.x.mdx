## Preview 1.107.04

2023년 8월 3일

<div class='indentTab'>

### `공통`

#### 대시보드

***트랜잭션 분석*** 창에서 다음의 내용을 수정합니다.

- <Status>Fixed</Status> <code>Container</code> 플랫폼의 마스터 프로젝트에서 진입 시 멀티 트랜잭션 정보 표시되지 않는 버그 수정

- <Status>Fixed</Status> <UI>멀티 트랜잭션</UI> 탭 > <UI>차트 보기</UI>의 프로젝트 필터 기능 오작동 수정

- <Status>Fixed</Status> 트랜잭션 목록의 트랜잭션 ID, 멀티 트랜잭션 ID 컬럼 정렬 오작동 수정

- <Status>Fixed</Status> <UI>트리 뷰</UI> 탭의 버튼 설명 툴팁 번역 누락 수정

  - **Display Time** → ***시간 표시***

  - **Hide Time** → ***시간 숨기기***

  - **Display Time Bar** → ***시간바 표시***

- <Status>Change</Status> 트랜잭션 목록의 검색 기능 대상 변경

    - 변경 전: 검색 대상 컬럼 지정이 필요

    - 변경 후: 화면에 표시되는 컬럼
