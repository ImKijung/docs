  **사용자 헤더**: key, value 쌍으로 입력한 정보를 웹훅 수신 시 헤더에서 확인 할 수 있습니다. 다음은 사용자 헤더의 key, value를 `User-Header-Key`, `TestValue`로 입력한 경우 웹훅 수신 정보에 대한 예시입니다.
        
  ```http {8} title='example'
  POST /test HTTP/1.1
  Host: whatap.requestcatcher.com
  Accept: text/plain, application/json, application/*+json, */*
  Connection: keep-alive
  Content-Length: 234
  Content-Type: application/json
  User-Agent: Java/17.0.7
  User-Header-Key: Test-Value
  ```
