* 배포일: 2022-12-28

* [https://preview.whatap.io](https://preview.whatap.io)에서 다음 추가한 신기능과 수정한 내용을 미리 경험할 수 있습니다. 

---

### 공통

#### 프로젝트 생성

- <Status>Change</Status> 메뉴 이름을 <UI>프로젝트 목록</UI>에서 <UI>상품 카탈로그</UI>로 변경

- <Status>Change</Status> 프로젝트 생성 url 경로 및 화면 UI 변경

#### 프로젝트 목록

- <Status>Fixed</Status> 클러스터 프로젝트 카드 뷰에서 에이전트 수 표시 오류 수정
    
- <Status>Fixed</Status> 즐겨찾기 기능이 오작동하는 현상 수정

- <Status>Change</Status> 리스트 뷰에서 프로젝트 항목 선택 시 새창이 아닌 페이지로 이동하도록 수정

#### 서버 상세, 노드 상세

<Status>Fixed</Status> 다음 메뉴 경로에서 '리소스 정보 자세히 보기' 클릭 시, 조회에 사용된 시간 값과 자세히 보기 창의 시간이 일치하지 않는 버그 수정

  - 서버 프로젝트 > <UI>서버 목록</UI> > <UI>서버 상세</UI>
  
  - 쿠버네티스 프로젝트 > <UI>리소스</UI> > <UI>노드 상세</UI>

#### 경고 알림 > 이벤트 설정

- <Status>Change</Status> <UI>이벤트 설정</UI> > <UI>메트릭스</UI> 탭에서 알림 목록을 제목 기준으로 정렬하도록 변경

- <Status>Fixed</Status> 프로젝트 수정 권한이 없는 사용자가 <UI>이벤트 설정</UI> 메뉴에서 알림을 수정할 수 없는 버그 수정

#### 분석 > 메트릭스 차트

- <Status>Change</Status> 위젯에서 에이전트를 다중 선택 가능하도록 기능 수정
    
- <Status>Feature</Status> 위젯 차트에 Y축의 최대값을 고정할 수 있도록 옵션 추가

- <Status>Fixed</Status> 위젯 ‘상세 보기’에서 차트 드래그 범위가 1분 이상인 경우에만 조회 기간을 변경하도록 수정

- <Status>Fixed</Status> 위젯 ‘상세 보기’에서 데이터 요청 로딩 중인 경우, 위젯의 조회 기간을 변경할 수 없도록 조회 버튼 비활성화

#### Flex 보드

- <Status>Fixed</Status> 애플리케이션 위젯의 제품군 지원 범위 수정

- <Status>Feature</Status> 로그 카운트 수 위젯 카테고리 선택 옵션에 전체 카테고리 옵션 추가

- <Status>Change</Status> 대시보드 템플릿 목록에서 각 항목마다 지원 가능한 제품 카테고리는 아이콘을 표시하도록 수정

#### 에이전트 / 서버 설정

<Status>Feature</Status> 다음 메뉴 경로에서 비활성화 에이전트에 대해 <b>삭제</b> 버튼, <UI>모든 비활성화된 에이전트 삭제</UI> 버튼 추가

* 애플리케이션, 쿠버네티스 프로젝트: ***관리*** > ***에이전트 설정***
* 서버 프로젝트: ***서버 설정*** > ***서버 설정***

### 쿠버네티스 프로젝트

#### 대시보드 > 컨테이너맵

- <Status>Fixed</Status> <UI>Ciritical</UI> 입력창과 <UI>Warning</UI> 입력창에 동일한 값을 입력하면 입력 창 증가하는 버그 수정
    
- <Status>Feature</Status> 상세 보기 화면에서 <code>ContainerId</code> Tag에 컨테이너 목록으로 이동하는 링크(<UI>컨테이너 성능 정보</UI>) 추가
    
- <Status>Change</Status> TX 프로파일 창 크기 조절
    
- <Status>Feature</Status> PodStatus가 Pending 상태일 때 <UI>이벤트</UI> 탭을 우선 표시하도록 기능 추가

- <Status>Fixed</Status> <UI>로그</UI> 탭 진입하면 컨텐츠가 깜빡이는 현상 수정

#### 리소스 > 컨테이너목록

- <Status>Fixed</Status> 컨테이너 목록의 갯수가 1200개일 경우 화면 멈춤 현상 수정
    
- <Status>Feature</Status> 컨테이너 선택 시에 나타나는 <UI>Memory</UI> 탭에 <b>Container Memory Total RSS By Limit (%)</b> 지표 추가
