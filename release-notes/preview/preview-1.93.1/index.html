2022-12-08​

Flex 보드​

Fixed 대시보드를 읽기 전용으로 복사한 경우 삭제 가능하도록 수정

로그​

  • Fixed 필터에 비교 연산자로 ‘==’ 입력 시, 표현식에 ==’가 중복으로 표시되는 버그 수정
  • Fixed 필터에 숫자 필드(.n 필드) 입력시, prefix로 ‘+’가 붙어서 검색이 되지 않는 버그 수정

결제 정보​

Change 결제 수단 수정 버튼 클릭 시 세금 계산서 발행일 옵션 추가

2022-12-05​

컨테이너 맵​

Fixed 에이전트 1.2.0 버전 업데이트 후 Pod 카테고리 선택시 ‘에이전트 업데이트’ 안내 메시지를 출력하는 버그 수정

2022-12-02​

멀티 트랜잭션 추적​

Fixed 분석 > 멀티 트랜잭션 추적 메뉴에서 일부 테이터 조회 시 차트가 그려지지 않는 버그 수정

트랜잭션 검색​

Fixed 분석 > 트랜잭션 검색 메뉴에서, 이전 프로젝트에서 조회한 데이터가 프로젝트 이동 시 표시되는 버그 수정

2022-11-30​

공통​

  • Change 프로젝트 생성 시 플랫폼 선택 화면에서 인티그레이션 아이템의 베타 표시 제거
  • Change 이벤트 설정 메뉴와 이벤트 수신 설정 메뉴에서 이벤트에 적용 중인 이벤트 수신 태그는 삭제할 수 없도록 수정

컨테이너맵​

  • New 쿠버네티스 제품군의 실험실 메뉴에 Pending Pod 기능 추가
  • Change 한 개의 Pod 내에 1개 이상의 컨테이너가 존재할 경우 컨테이너 정보를 노출하도록 수정
  • Feature Job형태의 Pod에 추가 로그 정보를 제공하기 위해 연관이벤트에 CronJob 추가

Flex 보드​

  • Feature 대시보드에 서버 제품군 Log RPS Board 템플릿 추가
  • Feature 위젯 템플릿 추가 * 초당 로그 카운트 수 * 로그 4xx, 5xx 건수 * 로그 요청 성공률 * 로그 응답시간

정비 계획​

Change 화면 상단 정비 계획 안내 문구 스타일 변경 및 버튼 위치 수정

로그​

Change 로그 설정 메뉴에서 데이터 유지기간 설정 기능을 ‘카테고리별 로그 데이터 유지기간’ 설정으로 수정

멀티 트랜잭션 추적​

Fixed 분석 > 멀티 트랜잭션 추적 메뉴에서 ‘차트 새창으로 띄우기’ 버튼 클릭 시 발생하는 화이트 아웃 현상 수정

알림 메시지​

New Application, Container, DB, Server 제품군에 신규 메뉴 사이트 맵 > 실험실 > 알림 메시지 추가

이벤트 설정​

New 경고 알림 > 이벤트 설정 > 메트릭스 메뉴에서 실험실의 알림 메시지 설정 메뉴 이동이 가능한 버튼 추가

인티그레이션​

인티그레이션 아이템 설치 개수 변경(무제한 → 1개)에 따른 메뉴 진입 경로 및 디자인 수정

  • Change 인티그레이션 아이템 메뉴 위치를 관리 하위 메뉴로 이동, 하나의 인티그레이션 아이템만을 설치, 설정 가능하도록 수정
  • Change 프로젝트 진입 시 인티그레이션 아이템 설치 현황에 따라 진입 경로 수정

    • 아이템이 설치된 경우: 아이템 종류에 맞는 대시보드로 이동

      • aws: Amazon CloudWatch > EC2 메뉴로 이동
      • ecs: Amazon ECS > Task 메뉴로 이동
      • 그 외: Flex 보드 메뉴로 이동

    • 아이템이 설치되지 않은 경우: 관리 > 인티그레이션 아이템 메뉴로 이동

  • Change 프로젝트 카드 디자인 수정
  • Change 인트그레이션 아이템이 ECS인 경우: 관리 > 인티그레이션 설치 메뉴에서 설치 안내 페이지 하단에 고정되었던 모니터링 서비스 활성화 버튼 위치를 설치 과정 내부로 이동

인티그레이션, 클라우드 모니터링​

Deprecate 관리 > 인티그레이션 아이템 메뉴에서 AWS CloudWatch 서비스 설정 시 전체 선택 체크 박스 제거

에이전트 설정​

Fixed 에이전트 이름을 별칭으로 설정하면 서버 목록 화면에 반영되지 않는 문제 수정

결제 관리​

결제 관리 > 유료 전환 내역 메뉴에서 다음의 기능을 수정합니다.
  • Change 월간 예상비용 컬럼에 ‘세션 사용 상세 내역’ 추가 표시하도록 수정
  • Change 제품 종류 컬럼에 MYSQL RDS 제품 표시하도록 수정
