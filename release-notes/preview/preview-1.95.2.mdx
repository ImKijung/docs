### 공통

#### Flex 보드

<Status>Feature</Status> <UI>홈 화면</UI> > <UI>Flex 보드</UI> 메뉴의 대시보드 목록에서 대시보드 공유(<img src='/img/ico-flexboard-share.svg' alt='공유 아이콘' />) 기능 추가

#### 분석

- <Status>Fixed</Status> <UI>사이트맵</UI> > <UI>분석</UI> > <UI>메트릭스 이상 탐지</UI> 메뉴에서 카테고리, 필드 선택 UI 버그로 인해 UI 로직 교체

- <Status>Fixed</Status> <UI>큐브</UI> 메뉴에서 조회가 끝나지 않은 경우 <UI>시작일</UI>, <UI>관심 지표</UI>, 대시보드 탭을 변경할 수 없도록 수정

#### 관리

***관리*** > ***프로젝트 멤버*** 메뉴에서 다음의 기능을 추가합니다.

- <Status>Feature</Status> 멤버 목록의 <UI>소속</UI> 컬럼에서 소속 유형이 그룹인지 프로젝트인지 표기 추가

- <Status>Feature</Status> 멤버 삭제(<img src='/img/ico-trash.svg' alt='삭제 아이콘'/>) 버튼 비활성화 시 툴팁(삭제 불가능한 이유) 추가


#### 트랜잭션 상세 `Application`

- <Status>Change</Status> <UI>SQL</UI> 분석 탭의 명칭을 <UI>SQL 요약</UI>으로 변경

- <Status>Change</Status> <UI>메소드 분석</UI> 탭의 명칭을 <UI>메소드 요약</UI>으로 변경

- <Status>Feature</Status> <UI>HTTP 호출 요약</UI> 탭 추가 : HTTP 호출의 호출 건수, 합계 시간, 평균 시간 정보  제공
