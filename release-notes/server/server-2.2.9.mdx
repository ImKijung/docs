## 버그 수정

### 윈도우 모니터링 에이전트 02시경 자동 재시작 오류 수정

2.2.8 버전의 코드 오류로 윈도우 모니터링 에이전트가 자동 재시작하는 오류가 있습니다. 오전 2시경 정지되고 Agent NoData 오탐이 발생할 수 있습니다. 2.2.8 버전을 사용하고 있을 경우 업데이트를 권장합니다.
