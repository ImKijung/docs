컨테이너 맵 상세 보기

컨테이너 맵 메뉴에서 선택한 대상에 대한 자세한 정보를 상세 보기 화면에서 확인할 수 있습니다.

  1. 컨테이너 맵 메뉴 화면에서 특정 대상(개별, 집합)을 선택하세요.

  2. 화면 오른쪽에 요약 보기 화면이 나타납니다.

  3. 오른쪽 위에 버튼을 선택하면 더욱 자세한 정보를 확인할 수 있습니다.

컨테이너 맵 상세 보기 화면은 다음과 같이 구성되어 있습니다.

  • 대상 카드 (컨테이너 또는 Pod)

  • 대상 리스트

  • 메뉴별 화면

    • 트레이스 > 대시보드, 트랜잭션 목록
    • 메트릭스
    • 로그 > 최근 로그 메시지, 로그 검색
    • 이벤트 > Kubernates, WhaTap
