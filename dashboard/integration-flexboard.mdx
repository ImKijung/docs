인티그레이션 프로젝트 Flex 보드의 진입 경로와 제공 기능은 다음을 참조하세요.

* 홈 화면 > 인티그레이션 프로젝트 선택 > ***인티그레이션 프로젝트*** 메뉴 > ***Flex 보드***
    * **사용자 정의형 대시보드 관리**  
        각 아이템이 수집하는 메트릭스 데이터를 기반으로 대시보드 화면을 구성할 수 있습니다.
        ![인티그레이션 프로젝트 - Flexboard](https://img.whatap.io/media/images/whatap_integration_project_flexboard_create.png)

    * **아이템별 대시보드 템플릿 제공**  
        인티그레이션 아이템의 각 대시보드 메뉴에 반영된 대시보드를 템플릿으로 제공하기 때문에, 이 템플릿 기반의 대시보드 레이아웃에 다른 위젯을 추가하거나 설정을 변경할 수 있습니다.   
        * Flex 보드 선택
            ![Flexboard 선택](https://img.whatap.io/media/images/whatap_integration_project_flexboard_management.png)
