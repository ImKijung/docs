## Application Log

애플리케이션 로그를 빠르고 쉽게 관리하세요. 로그 모니터링은 추가 설치 없이, 설정 하나로 로그를 통합 수집 관리합니다. 라이브 Tail로 흩어진 로그를 한 눈에 확인합니다.
