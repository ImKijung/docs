와탭의 애플리케이션 모니터링은 장애 재현 없이 실시간으로 모니터링이 가능합니다. '**액티브 스택**'이라는 와탭만의 특허 기술로 기존의 다른 제품에서는 발견하지 못했던 숨겨진 영역의 문제 원인을 찾아 개선 포인트를 제공합니다.

{@video: https://www.whatap.io/video/main-product/monitoring_step_0.mp4}

## 애플리케이션 모니터링 제공 기능

### 실시간 모니터링

- 특정 애플리케이션을 모니터링 하기 위한 필터링 기능
- 진행중 트랜잭션 정보
    - 지연 시간대 별 트랜잭션 수
    - 트랜잭션 프로파일
    - 진행 중 트랜잭션 중지
- 종료된 트랜잭션 정보
    - 응답시간 분포
    - 트랜잭션 프로파일
    - 트랜잭션 수행 중 확보된 스택 정보
- 애플리케이션 그룹 및 개별 애플리케이션의 TPS 및 응답시간 추이
- 5분간 집계된 사용자 수 추이
- CPU 및 Memory 사용량 추이

### 성능 분석

- 진행 중 트랜잭션에 대한 스택 정보
- 스택 전체 일치 통계
- 스택 마지막 정보 일치 기준 통계 및 호출 비중 통계
- 일간~5분 구간의 스택 통계 정보
- URL 단위의 스택 정보

### 이벤트 알림

- 이메일, 문자, SNS(Telegram, Slack) 발송
- 이벤트 지속 기간 및 임계치 설정에 의한 이벤트 발생
- 무음 설정 시간 등록에 의한 동일 이벤트 알림 억제 기능
- CPU, Memory, Disk 알림 기능
- 실시간 트랜잭션 개수 임계치 설정에 의한 알림
- 트랜잭션 에러 개수 임계치 설정에 의한 알림
- 응답시간 임계치 초과 트랜잭션 개수에 의한 알림
- 응답시간 분포의 가로 및 세로 분포 패턴에 의한 알림

### 통계 및 보고

- 일간/주간 보고서 : 실시간 사용자, TPS, 응답시간, CPU 추이 등
- 월간 보고서
    - 실시간 사용자, TPS, 응답시간, CPU 추이 등
    - 일간 방문자 수 추이, 월간 방문자 수, 고객 충성도 지표
- 애플리케이션 단위 일간 보고서 : 프로젝트 단위가 아닌 애플리케이션 단위의 보고서
- 애플리케이션 피크 성능 비교 보고서 : 전일, 전주 피크 타임과의 비교

### 관리 기능

- 권한 관리
    - super admin, admin, user의 차등화된 권한 설정 기능 제공
    - 사용자 초대 기능을 통한 권한 부여 기능
- 에이전트 업데이트
- 에이전트 옵션 설정
    - 프로파일 수집량 제어
    - 임계 트랜잭션 처리량 초과 시 별도 페이지 전환 및 에러 발생기능
    - 액티브 스택 수집 주기 변경
    - 상세 프로파일 수집 임계치 변경
