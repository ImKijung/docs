와탭 에이전트는 에이전트 별 필요한 설정을 *whatap.conf* 파일에 작성합니다. 에이전트는 환경변수를 통해 설정 파일의 위치를 파악하고 로딩합니다. 컨테이너나 PaaS 환경에서 Node.js 서버가 동작하는 경우 환경 변수를 통해 에이전트 설정을 구성합니다.
