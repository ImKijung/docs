구글 앱 엔진(Google Cloud App Engine)에 PHP 모니터링을 설치하기 위해서는 custom docker를 사용해야 합니다. 구글 앱 엔진 PHP의 기본형은 PHP 모니터링의 PHP Extension 및 Service (데몬)의 실행이 불가능합니다. Docker Container를 통해 설치하세요.
