1. 와탭 저장소(Repository)를 등록하세요.

    ```bash
    $ sudo rpm -Uvh http://repo.whatap.io/centos/5/noarch/whatap-repo-1.0-1.noarch.rpm
    ```

1. 다음 명령어를 통해 패키지를 설치하세요.

    ```bash
    $ sudo yum install whatap-php
    ```
1. `install.sh` 설정 스크립트를 실행하세요. 스크립트 실행으로 PHP 확장 모듈 설정과 whatap-php 서비스 데몬이 등록됩니다. 자세한 과정은 다음을 참고하세요.
