## 에이전트 연결 확인

[와탭 모니터링 서비스](https://service.whatap.io)로 이동하세요. 로그인 후 프로젝트 목록에서 생성한 프로젝트를 선택하세요. 화면에 에이전트가 나타나면 적용을 완료했습니다.

![프로젝트 화면](https://img.whatap.io/media/images/Screenshot_2020-12-03_test_14737_-_Application_Monitoring4.png)

:::note
* 에이전트 이름의 기본값은 {type}-{ip2}-{ip3}-{port}입니다.
* 에이전트의 이름을 변경하거나 속성을 추가하려면 [에이전트 이름 식별](java-agent-name)을 참조하세요.
:::
