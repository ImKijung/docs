* **profile_error_httpc_time_max** <span class='type'>Int</span><br/>
기본값 `10000`<br/>
HTTPC 수행 시간이 지정한 값을 초과하면 TOO SLOW 에러로 처리합니다. `0`으로 설정하면 에러 처리하지 않습니다.

* **profile_httpc_resource_enabled** <span class='type'>Boolean</span><br/>
기본값 `false`<br/>
프로파일에서 HTTP Call 스텝을 수집할 때 해당 스텝에서 사용한 CPU와 메모리 사용량을 추적합니다.

* **profile_position_httpc** <span class='type'>Boolean</span><br/>
기본값 `false`<br/>
HTTPC가 수행하는 시점의 StackTrace를 기록합니다.

* **trace_httpc_normalize_enabled** <span class='type'>Boolean</span><br/>
기본값 `true`<br/>
트랜잭션 내 HTTPC URL을 파싱해 정규화하는 기능을 활성화합니다.

* **trace_httpc_normalize_urls** <span class='type'>String</span><br/>
    정규화할 HTTPC URL 패턴을 설정합니다. 호출 URL 패턴을 파싱해 패스 파라미터를 제거합니다.
:::note
예시, **/a/{v}/b**라고 선언하면 **a/123/b** → **a/{v}/b**로 치환합니다. 여러 개를 등록할 때는 쉼표(,)를 구분자로 사용합니다. 치환 패턴 정리 후 보완이 필요합니다.
:::

* **hook_httpc_patterns** <span class='type'>String</span><br/>
HTTP outbound 호출을 수행하는 `full package 클래스명`.`메소드`를 설정합니다.
