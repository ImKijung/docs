## Node.js 버전

0.1.12 버전 이상

## 운영체제

Windows, mac(osx), linux, ubuntu, centos 등의 Node.js를 실행할 수 있는 환경

## 데이터베이스

MySQL, Microsoft SQL, MongoDB, Redis 등

## 애플리케이션 서버

Express, Hapi, Mean, Koa, Restify 등의 Node.js 기본 모듈 사용하는 모든 프레임워크
