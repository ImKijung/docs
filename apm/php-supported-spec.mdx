와탭 PHP 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요.

* **지원 버전** :

    * PHP extension module
    
    * PHP 5.2, 5.3, 5.4, 5.5, 5.6, 7.0, 7.1
    
    * ZTS(Zend Thread Safe) 지원


* **운영체제** :

    * CentOS/Red Hat 6.x (64bit) 이상
    
    * Debian/Ubuntu 12.04 (64bit) 이상
    
    * Alpine Linux (64bit)
    
    * FreeBSD 10.x (64bit) 이상

* **데이터베이스** :

    * Oracle(OCI8)
    
    * MySQL
    
    * SQL Server
    
    * PostgreSQL(psql)

* **지원 라이브러리** :

    * PDO
    
    * MySQLi
    
    * cURL

* **애플리케이션 서버** :

    * Apache
    
    * PHP-FPM(Nginx)
