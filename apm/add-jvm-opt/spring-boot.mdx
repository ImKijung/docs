Spring Boot를 사용하는 애플리케이션 서버 환경에서 JVM 옵션을 추가하는 방법을 확인하세요. 

시작 스크립트에 JVM 옵션으로 `-javaagent`를 추가하세요.

```bash
java -javaagent:{와탭 설치 경로}/whatap.agent-X.Y.Z.jar -jar {application.jar}
```
