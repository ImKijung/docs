### 프로젝트 목록 조회

<span class='type'>GET</span> open/api/json/projects

`api_key`는 계정 API 토큰을 입력하세요.

```bash title='curl'
$ curl -L -w "\n" -H "x-whatap-token: {{api_key}}" "https://api.whatap.io/open/api/json/projects"
```
