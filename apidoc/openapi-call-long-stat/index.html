장기 통계

액티브 사용자(1일 단위)​

1일간 집계된 고유 사용자 통계로 etime은 stime을 기준으로 31일을 초과할 수 없습니다.

GET https://api.whatap.io/open/api/json/visitor_d/stitme/etime
command
$ curl -L -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://api.whatap.io/open/api/json/visitor_d/1554163200000/1554422400000"
response
{
  "data": [
    [
      1554163200000, // 1일 단위 (24*60*60*1000)
      1007
    ],
    [
      1554249600000,
      1007
    ],
    [
      1554336000000,
      1007
    ]
  ],
  "pcode": 3000000079,
  "stime": 1554163200000,
  "etime": 1554422400000,
  "total": 3
}
