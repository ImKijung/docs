홈 화면 > 프로젝트 선택 > ***서버 목록*** > ***서버 상세***

와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 ***서버 목록*** 메뉴 하위에 ***서버 상세*** 메뉴를 선택하세요. 

***서버 상세***는 서버의 자원 현황과 실행 프로세스의 상세 정보를 제공합니다.

![서버 상세](/img/server-detail.png)

## ![](/img/number-01.png) 상단 옵션

상단 ![number 1](/img/number-01.png) 영역에서 데이터 갱신 시간 및 조회할 서버를 지정할 수 있습니다. 

### 데이터 갱신 시간

서버 상세 페이지의 데이터는 5초마다 자동으로 갱신합니다. 

<div style={{textAlign: 'center'}}>

![sc](/img/server-list-data-rn.png)
</div>

* ![일시정지 아이콘](/img/ic-s-pause.svg)***일시정지*** : 사용자는 데이터의 자동 갱신을 중지할 수 있습니다. 
* ![재생 아이콘](/img/ic-play-grn.svg)***재생*** : 사용자는 중지한 자동 갱신을 재시작할 수 있습니다.
* ![새로고침 아이콘](/img/ic-refresh2.svg)***새로고침*** : 사용자는 현재 시간을 기준으로 데이터를 갱신할 수 있습니다.
