#### 필터 적용  

필터를 적용하면 입력한 조건에 맞는 로그를 필터링합니다. 복수의 필터를 입력할 수 있습니다. 필터의 태그가 같은 경우 OR(`||`)로, 그렇지 않은 경우는 AND(`&&`)로 적용됩니다. 

![](https://img.whatap.io/media/images/livetail_003_filter.png)

입력창에 값을 직접 입력하거나 ***필터*** 입력창을 클릭해 필터를 지정할 수 있습니다. ![아이콘](/img/ic-search-on-press.svg) ***검색*** 버튼을 선택하면 필터가 적용된 데이터를 ![number-s3](/img/number-03.png) 영역에서 조회할 수 있습니다.

#### 필터 수정

필터에 값을 입력한 뒤 입력한 값을 클릭하면 해당 값을 수정할 수 있습니다.  
![](https://img.whatap.io/media/images/livetail_005_filter_input.png)

#### 필터 수식

입력된 필터 값 아래에 있는 수식(expression)은 로그 데이터 조회 시 적용될 필터 수식 미리 보기입니다.  
![](https://img.whatap.io/media/images/livetail_006_filter_expression.png)

#### 검색 키(Search Key)

다음 이미지에서 파란색 박스 부분은 파싱(parsing)된 검색 키입니다. 검색 키는 ***로그*** > ***로그 설정***의 ***로그 Parser 설정***에서 파싱 로직을 등록해 설정할 수 있습니다.  
![](https://img.whatap.io/media/images/livetail_007_parsed_key.png)

#### 필터 입력 문법

필터 입력 시 사용할 수 있는 문법 목록입니다. ***라이브 테일***과 ***로그 검색***은 동일한 문법을 사용합니다.

|문법|예시|설명|
|----|----|----|
|include(+)|+ote-gateway|ote-gateway를 포함한 것만 필터링|
|exclude(-)|-ote-gateway|ote-gateway를 제외한 것만 필터링|
|startWith(*)|ERROR*|ERROR로 시작하는 것만 필터링|
|endWith(*)|*ERROR|ERROR로 끝나는 것만 필터링|
