## Open API 호출

Open API 호출 시에는 계정 API 토큰 또는 프로젝트 코드와 프로젝트 API 토큰을 헤더로 전송합니다. 본 문서에서는 curl을 통한 호출 예시를 제시합니다.

### 프로젝트 목록 조회

```bash title='curl'
$ curl -L -w "\n" -H "x-whatap-token: {{api_key}}" "https://api.whatap.io/open/api/json/projects"
```
