## whatap output plugin

Telegraf에서 수집하는 메트릭스를 와탭 수집서버로 전달할 수 있습니다. 수집된 데이터는 whatap의 프로젝트 내부에서 확인 가능합니다.

:::note
elegraf 오픈 소스 내에 whatap plugin 리뷰가 진행 중입니다.
정식 배포 전에는 whatap plugin을 포함하는 Telegraf를 교체해서 사용하세요.
:::
