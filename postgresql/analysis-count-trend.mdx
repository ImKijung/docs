## 카운트 추이 비교 

홈 화면 > 프로젝트 선택 > ***분석*** > ***카운트 추이 비교***

<ProdImg img='db-compare-count-trend.png' desc='카운트 추이' />

서로 다른 날자의 카운트 추이를 비교할 수 있습니다.

마우스를 차트위로 마우스 오버해 툴팁을 통해서 값을 비교할 수 있음
지표를 추가 또는 변경 가능하고 위치 이동 가능
액티브 세션 영역에서 보는 방법
시간을 선택하고 세세하게 보는 방법, 드래그하거나

:::note

***액티브 세션*** 테이블 목록에 글자 색상은 검정색 → <span class='slow'>주황색</span> → <span class='vslow'>빨강색</span> 순으로 세션의 수행 속도가 느린 것을 의미합니다.

:::
