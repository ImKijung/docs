데이터베이스 모니터링 지원 환경

데이터베이스 모니터링을 시작하기 전에 지원하는 데이터베이스 종류에 대해서 알아보세요.

지원 데이터베이스 종류​

와탭 데이터베이스 모니터링 서비스(DBX)에서는 성능 분석을 위해 현재 진행 중인 스탯 지표, 세션 데이터를 수집하며 수집된 데이터들을 저장합니다. 특정 기간 동안의 전체적인 추이를 분석하거나 비교해볼 수 있습니다. 또한 특정 시점에 진행 중이었던 세션을 파악하여 느린 Query를 분석할 수 있습니다. 수집된 SQL들을 재 가공하여 SQL들의 통계 데이터 분석이 가능합니다.

DBX에서는 다음의 데이터베이스들을 지원합니다. Java 6 이상 JDK를 설치해야 합니다.

데이터베이스별 지원 버전​

  • Oracle 10g 버전 이상
  • Postgresql 9.2 버전 이상 / EDB PAS 9.2 버전 이상
  • MySQL 5.5 버전 이상 / MariaDB 5.5 버전 이상
  • MSSQL - SQL Server 2005 이상

브라우저 지원​

와탭 모니터링 서비스는 웹브라우저와 모바일 앱에서 이용할 수 있습니다.

브라우저권장여부지원버전
Google Chrome✅58 이상
Mozilla FireFox✅52 이상
Edge❌38.14393 이상
Safari❌Untested
노트

브라우저 호환성과 성능을 이유로 Chrome, FireFox 최신 버전 사용을 권장합니다.
사용자 인터페이스(User Interface, UI)는 HTML5 표준 기술로 구현하여 Internet Explorer는 지원하지 않습니다.

방화벽​

와탭 에이전트는 수집 서버 TCP 6600 포트로 접속 가능해야 합니다.

모니터링 대상과 가까운 수집 서버 주소를 허용하세요.
