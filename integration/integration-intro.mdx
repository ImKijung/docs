## 인티그레이션 아이템

인티그레이션 아이템은 인티그레이션 프로젝트 내에서 사용 가능한 모니터링 기능입니다. 현재 사용 가능한 주요 아이템은 다음과 같습니다.

* Amazon CloudWatch
* Amazon ECS
* Naver Cloud Monitoring
* Oracle Cloud Monitor
* Azure Monitor

각 아이템이 제공하는 설정 가이드에 따라 인티그레이션 아이템을 설치(활성화)하면, ***대시보드***를 비롯한 각종 ***분석*** 및 ***이벤트 설정*** 기능을 사용할 수 있습니다.
