## 인티그레이션 프로젝트 생성

인티그레이션 프로젝트 생성 방법은 다음과 같습니다.

1. ***프로젝트 추가*** 화면 내에서 ***인티그레이션***에 해당하는 대상을 선택하세요.
    ![인티그레이션 프로젝트 생성](https://img.whatap.io/media/images/whatap-integration-project-create.png)
1. 프로젝트 이름, 데이터 서버 지역 등 기본 정보 입력 후 ***저장*** 버튼을 눌러 프로젝트 생성을 완료하세요.
    ![세부 설치 진행](https://img.whatap.io/media/images/whatap-integration-project-install.png)
1. 이후 표시되는 내용에 따라 선택된 각 아이템에 대한 설치를 계속 진행하세요.
