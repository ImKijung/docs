## 에이전트 업데이트

에이전트 업데이트가 필요한 경우 다음 명령을 실행하세요.

* **마스터 에이전트**

  ```bash
  kubectl set image deployment/whatap-master-agent whatap-master-agent=whatap/kube_mon:<에이전트버전> -n whatap-monitoring
  ```

* **노드 에이전트**

  ```bash
  kubectl set image daemonset/whatap-node-agent whatap-node-agent=whatap/kube_mon:<에이전트버전> -n whatap-monitoring
  ```

:::note
와탭 쿠버네티스 에이전트의 최신 버전은 [hub.docker.com](https://hub.docker.com/r/whatap/kube_mon/tags)에서 확인할 수 있습니다.
:::
