:::

홈 화면 > 프로젝트 선택 > ***리소스*** > ***컨테이너 목록***

현재 프로젝트의 컨테이너의 상세 정보를 목록 형태로 확인할 수 있습니다. 

* 10초 주기로 실시간 데이터를 업데이트합니다. 

* 컨테이너 상태 요약, 비교, 필터 및 CSV 파일 다운로드 기능을 제공합니다.

  * 다운로드한 CSV 파일명은 다음과 같습니다. *kubernetes_contianer_list_{pcode}_{timestamp}.csv*

컬럼 선택 시 해당 컬럼 기준으로 정렬한 목록을 조회할 수 있습니다.

화면 좌측 상단에서 전체 컨테이너 개수와 상태별 컨테이너 개수를 확인할 수 있습니다.
