<InDoc product="mysql">

<ImgLang img='event-config-v2-db01.png' desc='이벤트 목록' />

제공하는 이벤트 템플릿은 다음과 같습니다. <Cmdname sid="ITM07578" className="uitext" /> 항목에 설정된 기본값은 사용자가 원하는 값으로 수정할 수 있습니다.

* ***Active Sessions***

  카테고리: [`db_mysql_counter`](metrics-data-list)

  쿼리를 실행 중인 세션 수(`active_sessions`)가 30개를 초과하면 경고(<span class="slow">Warning</span>) 수준의 알림을, 50개를 초과하면 위험(<span class="vslow">Critical</span>) 수준의 알림을 보냅니다.

* ***Lock Wait Sessions***

  카테고리: [`db_mysql_counter`](metrics-data-list)

  현재 row lock을 획득하기 위해 대기 중인 수(`lock_wait_sessions`)가 3개를 초과하면 경고(<span class="slow">Warning</span>) 수준의 알림을, 5개를 초과하면 위험(<span class="vslow">Critical</span>) 수준의 알림을 보냅니다.

* ***Memory***

  카테고리: [`db_mysql_counter`](metrics-data-list)

  메모리 사용률(`mem(xos)`)이 80%를 초과하면 경고(<span class="slow">Warning</span>) 수준의 알림을, 95%를 초과하면 위험(<span class="vslow">Critical</span>) 수준의 알림을 보냅니다.

  :::note

  이 템플릿은 XOS 에이전트를 설정하고 실행했을 경우에만 사용할 수 있습니다. XOS 에이전트 설정에 대한 자세한 내용은 [다음 문서](agent-xos-settings)를 참조하세요.

  :::

* ***Replication Delay***

  카테고리: [`db_mysql_counter`](metrics-data-list)

  Replication 구조일 경우 replication db에 반영되는 지연시간(`replication_delay`)이 50초를 초과하면 경고(<span class="slow">Warning</span>) 수준의 알림을, 100초를 초과하면 위험(<span class="vslow">Critical</span>) 수준의 알림을 보냅니다.

</InDoc>
