홈 화면 > 프로젝트 선택 > ***대시보드*** > ***브라우저 에러***

***브라우저 에러*** 대시보드에서는 최종 사용자가 웹 애플리케이션을 사용하면서 브라우저에서 발생한 에러 데이터를 수집, 재가공한 내용을 위젯으로 제공합니다. 오류가 발생한 페이지를 찾고 원인을 분석하는데 도움을 받을 수 있습니다. 다양한 최종 사용자 환경을 통해 브라우저별, 기기별, OS별 에러 원인을 확인하세요.

:::note
* 각 위젯에서 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 선택하면 해당 위젯 데이터를 전체 화면 크기로 확인할 수 있습니다.
* 위젯의 오른쪽 위에 ![설정 아이콘](/img/ico-setting.svg) 버튼을 선택하면 차트 설정 화면이 나타납니다. 차트에 표시하는 옵션을 설정할 수 있습니다.
* 웹 브라우저의 화면 크기에 따라 위젯 오른쪽 위에 아이콘의 모양이 다를 수 있습니다. ![더보기 아이콘](/img/ico-more.svg) 버튼을 선택하면 추가 아이콘을 확인할 수 있습니다.
* 위젯 설정에 대한 자세한 내용은 [다음 설명서](../dashboard/dashboard-widget-setting)를 참조하세요.
:::

## ***에러 타입별 에러 건수***

![에러 타입별 에러 건수](/img/rum-browser-err-count.png)

와탭에서 정의한 브라우저 에러를 기준으로 에러 건수를 실시간으로 확인할 수 있습니다. 위젯 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 선택한 다음 시간대를 설정해 에러 건수를 조회할 수 있습니다. 

:::note
브라우저 에이전트에서 수집하는 에러 데이터에 관한 자세한 내용은 [다음 설명서](rum-collect-data#error-event)를 참조하세요.
:::

## 사용자 환경에 따른 에러 건수

![브라우저별 에러 건수](/img/rum-browser-err-browser.png)

![기기별 에러 건수](/img/rum-browser-err-device.png)

![OS별 에러 건수](/img/rum-browser-err-os.png)

브라우저별, 기기별, OS별 에러 건수를 실시간으로 확인할 수 있습니다. 위젯 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 선택한 다음 시간대를 설정해 에러 건수를 조회할 수 있습니다. 

* 각 에러는 특정 환경과 페이지 그룹 단위로 집계합니다. 위젯 차트에 마우스 오버하면 특정 환경, 페이지 그룹에서 발생하는 에러 정보를 툴팁 형태로 제공합니다.
