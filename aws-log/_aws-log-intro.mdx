AWS 리소스 로그는 S3 혹은 CloudWatch Log Group에 쌓입니다. 일반적으로 사용자는 S3와 Log Group에 각각 들어가 리소스별로 확인하는 번거로운 과정을 거쳐야 합니다. 

와탭 AWS Log를 활용한다면 사용자는 위의 번거로운 과정을 거칠 필요가 없습니다. AWS 리소스 로그가 쌓이는 S3와 CloudWatch Log Group을 와탭 AWS Log가 구독해 로그 생성 시 와탭 환경으로 보냅니다. 

![sc](/img/aws-log-intro.png)

* 리소스 로그를 한 번에 모아볼 수 있습니다.
* 시간대별 로그 발생량을 계산할 수 있습니다.
* 로그 내용으로 검색할 수 있습니다.
