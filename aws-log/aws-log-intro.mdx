와탭 AWS Log는 AWS Resource들의 로그를 수집하여 WhaTap 환경에 보냅니다.

![](https://img.whatap.io/media/images/fowarder.png)

:::note
와탭 AWS Log는 AWS Resource들의 로그가 쌓이는 S3, CloudWatch Log Group을 구독해 로그 생성 시 WhaTap 환경으로 보내줍니다.
:::
