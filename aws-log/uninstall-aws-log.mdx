AWS Log 프로젝트에서 사용하는 AWS 자원을 완전히 삭제하기 위해서는 다음 3가지 종류의 자원을 모두 삭제해야합니다.

* **CloudFormation**으로 생성된 **AWS 자원**
* **CloudWatch LogGroup**에 추가된 **Subscription filters**
* **S3 Bucket**에 추가된 **Event Notifications**

## ![](/img/number-01.png) CloudFormation으로 생성된 AWS 자원 삭제

![cloudformation 자원](/img/aws-log-cloudformation-resource.png)

1. AWS 콘솔에서 CloudFormation을 선택한 후 삭제할 Stack을 선택하세요.

1. Delete 버튼을 선택해 해당 Stack에 의해 생성된 모든 자원들 한 번에 삭제하세요.

## ![](/img/number-02.png) CloudWatch LogGroup의 Subscription filters 삭제

1. AWS 콘솔에서 CloudWatch를 선택하세요. 

1. 구독한 LogGroup을 찾습니다.

1. LogGroup의 `Subscription filters`에서 설정된 내역을 삭제하세요.

## ![](/img/number-03.png) S3 Bucket의 Event Notifications 삭제

1. AWS 콘솔에서 S3를 선택하세요. 

1. 구독한 S3 Bucket을 찾습니다.

1. Bucket의 `Properties`에서 설정된 **Event notifications** 내역을 삭제하세요.
