홈 화면 > 프로젝트 선택 > ***경고 알림*** > ***이벤트 설정*** > ***메트릭스*** 탭 선택

프로젝트 멤버에게 메일 또는 서드 파티 플러그인을 통해 전달하는 경고 알림 메시지를 수정할 수 있는 사용자 정의 기능을 제공합니다. 화면 오른쪽 위에 ***알림 메시지 설정*** 버튼을 선택하세요. 내용이 너무 길거나 많은 메시지의 경우 불필요하다고 생각되는 항목을 알림 메시지에서 감추거나 사용자가 원하는 특별한 내용을 추가할 수 있습니다.

![알림 메시지 설정](/img/notification-set-message.png)

:::note

* 이 기능은 **메트릭스**, **복합 메트릭스** 경고 알림에만 적용할 수 있습니다.

* 이 기능은 프로젝트 선택 > ***사이트맵*** > ***실험실*** > ***알림 메시지 설정*** 메뉴 경로로도 진입할 수 있습니다.

* 이 기능은 알림 설정 권한을 가진 멤버가 설정할 수 있습니다. 자세한 내용은 [다음 문서](../project/project-structure#project-auth)를 참조하세요.

:::

## 기본 화면 안내

![알림 메시지 설정](/img/notification-set-message-main.png)

* 알림 메시지의 종류는 <span class='occur set-noti'>발생</span>과 <span class='resolve set-noti'>해결</span>로 분류할 수 있습니다. <span class='occur set-noti'>발생</span>은 이벤트가 발생했을 때 전달되는 알림이며, <span class='resolve set-noti'>해결</span>은 이벤트가 해소되었을 때 전달되는 알림입니다.
