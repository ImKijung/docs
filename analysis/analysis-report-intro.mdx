<!-- 분석과 통계 -->

***프로젝트 메뉴*** → ***분석*** 

***프로젝트 메뉴*** → ***통계/보고서*** 

와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 ***프로젝트 메뉴*** 하위 메뉴를 선택하세요. ***분석*** 메뉴는 메트릭스 차트, MSA 분석, 히트맵, 스택, 큐브, 트랜잭션 추적 등을 제공합니다. ***통계/보고서*** 메뉴는 보고서와 통계를 제공합니다.
