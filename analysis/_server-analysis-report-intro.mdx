와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 ***프로젝트 메뉴*** 하위 메뉴를 선택하세요. 

* ***프로젝트 메뉴*** > ***분석***  
    ***분석*** 메뉴는 **메트릭스 차트**와 **큐브**를 통해 분석을 제공합니다.

* ***프로젝트 메뉴*** > ***서버 목록***  
    ***서버 목록*** 메뉴는 **서버 목록**과 **서버 상세** 내용을 제공합니다.   

* ***프로젝트 메뉴*** > ***보고서***  
    ***보고서*** 메뉴는 **주간 및 일간과 월간 보고서**를 제공합니다.
