홈 화면 > 프로젝트 선택 > ***서버 목록*** > ***서버 목록***

와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 프로젝트 메뉴 하위에 ***서버 목록*** > ***서버 목록*** 메뉴를 선택하세요. 

***서버 목록***은 수집 서버의 전체 현황을 나타내며 서버 관리기능 및 서버 간 비교 기능을 제공합니다.

![](/img/server-list.png)

## ![](/img/number-01.png) 서버 상태

![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-_Server_Monitoring24.png)

수집 서버의 상태별 현황을 나타냅니다. 상태는 1분 간격으로 갱신합니다.
* **비활성** : 에이전트가 해당 서버의 데이터 수집이 불가한 상태입니다.
* **위험** : 수집 서버에서 위험 이벤트가 발생한 상태입니다.
* ![아이콘](/img/ic-warning-filled.svg)**경고** : 수집 서버에서 경고 이벤트가 발생한 상태입니다.
* ![아이콘](/img/ic-check-circle-filled.svg)**정상** : 에이전트가 수집 서버에서 정상적으로 작동 중인 상태입니다.
* **일시정지** : 에이전트에서 데이터 수집을 일시적으로 중지한 상태입니다.

## ![](/img/number-02.png) 데이터 갱신
서버 목록 페이지의 데이터는 60초마다 자동으로 갱신합니다.

![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-_Server_Monitoring25.png)  
* ![아이콘](/img/ic-s-pause.svg)***일시정지*** : 사용자는 데이터의 자동 갱신을 중지할 수 있습니다. 
* ![아이콘](/img/ic-s-play-grn.svg)***재생*** : 사용자는 중지한 자동 갱신을 재시작할 수 있습니다.
* ![아이콘](/img/ic-refresh2.svg)***새로고침*** : 사용자는 현재 시간을 기준으로 데이터를 갱신할 수 있습니다.

## ![](/img/number-03.png) 서버 현황
에이전트가 할당된 전체 서버의 현황을 나타냅니다.

* ![number-s](/img/number-04.png) 영역 왼쪽 메뉴에서 ***컬럼 선택*** 버튼으로 테이블에 표시할 컬럼을 선택할 수 있습니다.
