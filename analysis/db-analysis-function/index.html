데이터베이스 분석하기

홈 화면 > 프로젝트 선택 > 분석

데이터베이스의 성능을 실시간 모니터링을 통해 분석할 수 있는 지표들에 대한 설명입니다.

카운트 추이​

홈 화면 > 프로젝트 선택 > 분석 > 카운트 추이

카운트 추이 메뉴에서는 다음 정보를 확인할 수 있습니다.

  • 하루 동안의 데이터베이스 운용 추이

  • 특정 시간대의 핵심 지표

  • 액티브 세션

    화면 아래 액티브 세션 탭을 선택하세요. 현재 진행 중인 세션들의 정보(Query 포함)를 확인합니다. 10초 이상 진행되고 있는 세션인 경우 노란색 , 20초 이상의 세션인 경우 빨간색으로 표현합니다. Query의 플랜 정보를 함께 확인할 수 있습니다.

  • 락 트리

    화면 아래 락 트리 탭을 선택하세요. 락에 대한 정보를 확인할 수 있습니다. holder와 waiter의 구조를 트리 형태로 표현하여 제공합니다. 10초 이상 진행되고 있는 세션인 경우 노란색 , 20초 이상의 세션인 경우 빨간색으로 표현합니다.

  • 프로세스 정보

    화면 아래 프로세스 정보 탭을 선택하세요. XOS 에이전트를 설치 한 경우 데이터베이스 서버의 프로세스에 대한 정보를 확인할 수 있습니다.

카운트 추이 비교​

홈 화면 > 프로젝트 선택 > 분석 > 카운트 추이 비교

서로 다른 날짜의 카운트 추이를 비교할 수 있습니다. 기준 날짜와 비교 날짜를 설정한 다음 버튼을 선택하세요.

카운트 추이 차트 활용하기​

카운트 추이 관련 메뉴에서 다음의 기능을 활용할 수 있습니다.

화면 오른쪽 위에 버튼을 선택하세요. 레이아웃 설정 창에서 1 x 3으로 설정한 다음 저장 버튼을 선택하세요. 시간은 최근 24시간으로 설정하세요. 다음과 같이 하루 동안의 데이터베이스 핵심 지표를 자세히 확인할 수 있습니다.

  • 지표 차트의 특정 시점을 클릭하면() 해당 지점을 붉은 선으로 표시하며 당시에 수집된 액티브 세션을 확인할 수 있습니다.

  • 액티브 세션 데이터는 5초 단위로 수집하고 있습니다. 화면 아래에 초 단위 셀을 선택해 5초 단위로 데이터를 확인할 수 있습니다.

  • 1분 단위로 시간을 이동하려면 또는 버튼을 선택하세요.

  • 상세 정보 데이터 셀에 표시하는 항목을 더하거나 빼려면 버튼을 선택하세요. 컬럼 편집 창이 나타면 원하는 항목을 더하거나 뺄 수 있습니다.

  • 버튼을 선택하면 데이터 셀 헤더에 필터를 적용해 원하는 데이터만 정렬할 수 있습니다.

  • 화면 아래에 출력된 데이터를 csv 파일로 다운로드할 수 있습니다. 버튼을 선택하세요.

락 트리​

홈 화면 > 프로젝트 선택 > 분석 > 락 트리

락에 대한 정보를 확인할 수 있습니다. holder와 waiter의 구조를 트리 형태로 표현하여 제공합니다. 락 트리 분석에 대한 자세한 설명은 다음 동영상을 참조하세요.

데드락​

홈 화면 > 프로젝트 선택 > 분석 > 데드락

데이터베이스에서 발생한 데드락에 대한 정보를 확인할 수 있습니다. holder와 waiter의 구조를 트리 형태로 표현하여 제공합니다.

노트

이 기능은 Mysql, MariaDB를 사용하는 플랫폼에서만 지원합니다.

병렬 쿼리 트리​

홈 화면 > 프로젝트 선택 > 분석 > 병렬 쿼리 트리

병렬 쿼리(Parallel Query) 트리의 정보를 트리 구조로 확인할 수 있습니다. Coordinator와 Slave의 구조를 트리 형태로 표현하여 제공합니다.

노트

이 기능은 Oracle, Tibero를 사용하는 플랫폼에서만 지원합니다.

데이터베이스 파라미터​

홈 화면 > 프로젝트 선택 > 분석 > 데이터베이스 파라미터

데이터베이스의 파라미터 값을 날짜별로 비교할 수 있습니다. 기본적으로 현재 시간의 데이터를 볼 수 있습니다. 날짜에서 과거 날짜를 선택한 다음 버튼을 클릭하세요. 현재 시간과 선택한 과거 날짜와의 변경된 사항을 확인하려면 View Changes 버튼을 선택하세요.

데이터베이스 파라미터 메뉴에 대한 자세한 내용은 다음 동영상을 참조하세요.
