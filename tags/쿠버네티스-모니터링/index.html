description

Event

와탭 쿠버네티스 모니터링에서는 쿠버네티스 이벤트, 와탭 이벤트를 제공합니다.

Log

컨테이너 또는 컨테이너 내부에서 실행 중인 애플리케이션이 출력하는 로그를 바로 확인할 수 있습니다.

Merics

쿠버네티스의 리소스 사용량을 확인하고, 사용량이 적절히 설정되었는지 확인할 수 있습니다.

MSA 분석

MSA 분석 기능은 와탭의 MTRACE(멀티 서버간 추적)기능을 통해서 수집된 트랜잭션들 사이의 호출 비중이 어떠한지를 분석합니다.

OOM Killed 컨테이너

Out Of Memory(OOM)가 발생한 컨테이너를 빠르게 검색하고 해당 시점의 메모리 사용량의 변화 추이를 확인할 수 있습니다.

POD 초기화 성능

POD 초기화 성능 기능을 활용하면 Pod 시작시 피크 사용량과 초기화 시간을 분석하여 최적의 Initial Delay 설정에 도움을 받을 수 있습니다.

Trace

와탭 쿠버네티스 모니터링의 트레이스 기능은 컨테이너 상에서 실행 중인 애플리케이션 상태 모니터링 정보를 제공합니다.
