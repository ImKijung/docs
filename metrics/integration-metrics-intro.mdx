메트릭스는 모니터링 대상의 성능 정보를 표현하는 시계열 데이터입니다. 인티그레이션 아이템 중 클라우드, Amazon ECS를 사용하면 메트릭스 데이터가 수집됩니다. 수집 후 다음과 같은 ***메트릭스*** 메뉴의 기능을 통해 통합 분석 기능을 이용할 수 있습니다.

* 메트릭스 큐브
* 메트릭스 차트
* 메트릭스 이상 탐지

## 메트릭스 카테고리

인티그레이션 아이템별 메트릭스는 카테고리 이름으로 구분이 가능합니다.

|아이템|카테고리 접두사|
|----|----|
|Amazon CloudWatch|aws_|
|Amazon ECS|ecs_|
|Naver Cloud Monitoring|ncloud_|
|Oracle Cloud Monitor|oci_|
|Azure Monitor|azure_|

## 메트릭스 큐브

![인티그레이션 프로젝트 - 메트릭스 큐브](https://img.whatap.io/media/images/whatap_integration_project_metrics_metrics_cube.png)

메트릭스 큐브는 각 카테고리의 메트릭스 필드 데이터의 추이 분석 기능을 제공합니다. 주요 기능은 다음과 같습니다.

* 카테고리 선택: 선택한 카테고리에 포함된 필드로 대시보드 화면을 구성합니다.
* 필터링: 태그 기반 필터링이 가능합니다. 
* 그룹화 옵션: 차트 데이터의 그룹화 속성 변경이 가능합니다.

## 메트릭스 차트

![인티그레이션 프로젝트 - 메트릭스 차트](https://img.whatap.io/media/images/whatap_integration_project_metrics_metrics_chart.png)

메트릭스 차트는 단일 메트릭스 조회 기능을 제공합니다. 주요 기능은 다음과 같습니다.

* 위젯 추가: 좌측 목록을 통해 필드를 선택하여 화면에 위젯을 추가할 수 있습니다.
* 개별 정보 조회: 상세 보기를 클릭해 해당 위젯의 개별 시리즈 정보를 조회할 수 있습니다.
