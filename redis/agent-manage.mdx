## 에이전트 업데이트

홈 화면 > 프로젝트 선택 > ***관리*** > ***에이전트 업데이트***

![에이전트 업데이트](/img/db-update-agent.png)

에이전트를 개별로 업데이트하려면 각 에이전트 항목의 오른쪽에 ***최신 버전*** 버튼을 선택하세요. 모든 에이전트를 한번에 업데이트하려면 ***최신 버전으로 모든 에이전트에 다운로드*** 버튼을 선택하세요.

에이전트가 위치한 서버에 자동으로 다운로드합니다. 다운로드를 완료하면 에이전트를 다시 시작하세요. 최신 버전의 에이전트로 자동 적용합니다.

## 에이전트 삭제

### 에이전트 설치 서버에서 삭제하기

<Tabs>
<TabItem value='linux' label='Linux' default>

서버 상에서 기동되는 Java 프로세스 중 `whatap.agent.dbx` 문자열을 포함한 프로세스를 종료합니다. 에이전트를 설치한 경로에서 *stop.sh*를 실행하세요. 이후 에이전트 파일 및 에이전트에 등록한 User 권한을 삭제 처리하세요.

</TabItem>
<TabItem value='windows' label='Windows'>

서버 상에 기동되는 Java 프로세스 중 에이전트 파일명의 jar 파일로 기동 중인 프로세스를 찾아 중지하세요. 만약 bat 파일을 Foreground에서 실행 중인 경우 해당 bat 실행 스크립트를 종료하세요. 이후 에이전트 파일 및 에이전트에 등록한 User 권한을 삭제하세요.

</TabItem>
</Tabs>


### 서비스 화면에서 삭제하기

[와탭 모니터링 서비스](https://service.whatap.io)에서 에이전트를 삭제하려면 ***에이전트 목록*** 메뉴로 이동하세요. 에이전트를 삭제하기 전에 에이전트를 설치한 서버에서 에이전트 프로세스를 종료해야 합니다. 

1. ***프로젝트 목록***에서 에이전트를 삭제할 프로젝트를 선택하세요.

1. 왼쪽 메뉴에서 ***관리*** > ***에이전트 목록***을 선택하세요.

1. ***에이전트 목록***에서 삭제하려는 에이전트의 가장 오른쪽에 ![삭제 아이콘](/img/ico-trash.svg) 버튼을 선택하세요.

:::note

* 비활성화 상태의 에이전트만 삭제할 수 있습니다.

  ![](/img/db-disable-agent.png)

* 에이전트 버전에 따라 삭제 기능을 지원하지 않을 수 있습니다. 최신 버전의 에이전트로 업데이트하세요.

:::
